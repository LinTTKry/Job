# 17.680. 验证回文字符串 Ⅱ

给定一个非空字符串 `s`，**最多**删除一个字符。判断是否能成为回文字符串。

**示例 1:**

```
输入: "aba"
输出: True
```

**示例 2:**

```
输入: "abca"
输出: True
解释: 你可以删除c字符。
```

```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
var validPalindrome = function(s) {
    let l = 0, r = s.length-1;
    while(l<r){
        if(s[l]==s[r]){
            l++;
            r--;
        }else{
            return validPalind(s,l+1,r)||validPalind(s,l,r-1)
        }
    }
    return true
};
function validPalind(s,l,r){
    while(l<r){
        if(s[l]!=s[r]){
            return false
        }else{
            l++;
            r--;
        }
    }
    return true
}
```
