# 01.爬楼梯

假设你正在爬楼梯。需要 _n_ 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**注意：**给定 _n_ 是一个正整数。

```javascript
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
   f = [n+1];
   f[1] = 1;
   f[0] = 1;
   for(let i = 2; i <= n; i++){
      f[i] = f[i-1]+f[i-2]
   }
   return f[n]
};
```
