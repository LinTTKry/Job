# 05. 最大子序和 53

给定一个整数数组 `nums` ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**示例 1：**

```
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 
```

```
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxSubArray = function(nums) {
   let max = nums[0];
   let f = nums[0];
   for(let i = 1; i < nums.length;i++){
      f = Math.max(f+nums[i],nums[i]);
      max = Math.max(max,f)
   }
   return max
}
```
